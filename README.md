# Simple-CRUD-API

### Rename ".env.example" to ".env"
### Install packages
```javascript
npm install
```
### Run in development mode
```javascript
npm run start:dev
```
### Run in producion mode
```javascript
npm run start:prod
```
### Run in development mode with load balancer
```javascript
npm run start:multi
```
### Run tests
```javascript
npm run test
```
